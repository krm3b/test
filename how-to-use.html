How to use

1

アバターを作成しよう

  • 好きなアバターを設定
  • 名前は「LINE名 フェーズ」
  • 初心者は🥲をつける
    例：「🥲LINE名 模写初級」

集まった仲間でこんな遊びをしていることも！

2

アバターを動かしてみよう

  • 矢印キーで移動
  • ダブルクリックで希望場所に飛ぶ
  • ショートカットキー利用

C → 車に乗る

X → 踊る

Z → 燃える

3

フロアを移動、交流しよう

  • 集中して作業したい
  • 誰かにちょっと質問したい
  • セミナーを聞きたい…etc

誰かに近づいたり、作業スペースなどの特定のエリアに入ると、そこにいる人たちと会話ができます。表示はアバターでも、カメラをオンで顔出ししても、自分のスタイルで参加できます！

Floor

作業フロア

モクモクルーム

作業に集中したい時に利用
会話はチャットのみの部屋もあります

会議室

各種講習会が実施されています

その他

思い思いの場所で、自分のペースで作業。会話しなくても、仲間の存在がやる気につながります。
時には雑談したり、先輩ユーザーからヒントをもらうことも！

屋上フロア

毎週月曜日に開催されるセミナーはこの屋上フロアが会場です。毎回100名近い参加者が集まり、知識を深め合っています。
